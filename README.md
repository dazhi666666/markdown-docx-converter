# Markdown 转 DOCX 转换器

一个 Python 工具，用于将包含 Markdown 格式的 `.docx` 文件转换为格式化的 Word 文档。

---

## 功能
- **支持的 Markdown 语法**：
  - 标题（`#`, `##`, `###` 等）
  - 加粗（`**文本**`）
  - 斜体（`*文本*`）
  - 列表（`* `, `- `）
- **可自定义样式**：
  - 标题（1-6 级）
  - 项目符号列表
  - 基本文本格式（加粗、斜体）
- **可扩展性**：
  - 易于添加对其他 Markdown 功能的支持（如链接、表格、代码块）。
